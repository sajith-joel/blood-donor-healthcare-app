# Assets Folder

Place your images here:
- logo.png (512x512) - App logo
- splash.png (1242x2436) - Splash screen
- icon.png (1024x1024) - App icon
- donor-avatar.png (200x200) - Default donor avatar
- hospital-icon.png (200x200) - Default hospital icon

For now, you can use placeholder images or empty files.
