# Blood Donor App

A React Native mobile application that connects hospitals with blood donors in real-time based on location and blood group requirements.

## Features

- **User Authentication** (Donor/Hospital roles)
- **Location-based blood requests** (3km for common blood, 5km for rare blood)
- **Real-time notifications** to nearby donors
- **Blood request management** for hospitals
- **Donation history tracking**
- **Push notifications** for urgent requests

## Tech Stack

- React Native (Expo)
- Firebase (Auth, Firestore, Functions, Cloud Messaging)
- React Navigation
- Expo Location Services
